ℹ️ What is UNVT?

The United Nations Vector Tile Toolkit (UNVT) is a collection of Open Source Software (OSS) to produce, host, style and optimize vector tiles for web mapping.

ℹ️ Projects

We have various projects under the UNVT because UNVT is a toolkit for supporting web map publishers. There are different deployment arrangement for different data environment. Still we are sharing software and documents as much as possible.

👷‍♂️ equinox

equinox is UNVT one-line installer for Raspbian Lite. If you use equinox, you do not have to work with docker. If you need a compact development or testing environment of UNVT on a Raspberry Pi, equinox is your solution.

👷‍♂️ nanban

nanban is a developer friendly UNVT Dockerfile based on (Intel) Ubuntu. This image can be used for the development purpose. If you want a user-friendly UNVT environment on your PC with Docker, nanban is your solution.

👷‍♂️ Vector Tile Academy (VTA) and naru

Vector Tile Academy (VTA) is a technology transfer material of UNVT. We are also implementing naru which is the working file repository for VTA, based on equinox in principle.
